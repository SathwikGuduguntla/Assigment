
## Library Member – Functional Verification

### 1️⃣ Create Member

**Steps:**

1. Navigate to **Library Management → Library Member → New**
2. Fill in all mandatory fields
3. Click **Save**
4. Verify that **member_id** is automatically generated (e.g., `LIB-MEM-00001`)

**Screenshot:**

![Create Member](Screenshot%20from%202026-03-07%2022-17-02.png)

---

### 2️⃣ Edit Member

**Steps:**

1. Open an existing **Library Member**
2. Change **Membership Type** to **Premium**
3. Click **Save**
4. Verify that the updated value is stored correctly

**Screenshot:**

![Edit Member](Screenshot%20from%202026-03-07%2023-11-57.png)

---

### 3️⃣ Unique Email Validation

**Steps:**

1. Create a new member with a specific email address
2. Attempt to create another member using the **same email**
3. Verify that the system shows a **validation error**

**Screenshot:**

![Unique Email Validation](Screenshot%20from%202026-03-07%2022-01-12.png)

---

### 4️⃣ List View Verification

**Steps:**

1. Create at least **three members**
2. Open the **Library Member List View**
3. Verify all records are displayed
4. Test the following features:

   * **Search functionality**
   * **Filters**
   * **Sorting**
